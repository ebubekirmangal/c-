İstediğimiz veriyi ve verinin türünü girdiğimiz ve sonra veriyi istenilen tarzda depolayan bir uygulamadır.(Excel gibi)

Backup, Restore ve Attach, Detach İşlemleri

Backup: Veritabanlarınızı korumanın ve veri kaybını önlemenin önemli bir yoludur.
Restore: Önceden yedeklenmiş verilerin geri yüklenmesini ifade eder. Veri kaybını önlemek ve iş sürekliliğini sağlamak için yapılan yedekleme işlemi sonucunda oluşturulan veri kopyası, "restore" işlemi ile veri tabanına geri dönüştürülür.
Attach: Bir veritabanının SQL Server örneğine eklenmesini sağlar. Bu işlem, bir veritabanının var olan bir SQL Server örneğine yüklenmesini ve kullanılabilir hale getirilmesini sağlar.
Detach: Bir SQL Server veritabanının mevcut örneğinden kaldırılmasını sağlar. Bu işlem, veritabanını SQL Server örneğinden ayırarak veritabanını kullanılamaz hale getirir.

Data Definition Language - Veri Tanımlama Dili

Create: Yeni veritabanı nesnelerini oluşturmak için kullanılır.
Alter: Veritabanı nesnelerini değiştirmek için kullanılır.
Drop: Var olan bir veritabanı nesnesini silmek için kullanılır.

SQL
// Querry üzerinden Data Base oluşturma
CREATE DATABASE Test
GO

// Querry üzerinden Data Base silme
DROP DATABASE Test

// Querry üzerinden Tablo Oluşturma
CREATE TABLE Employees(
    Id İnt NOT NULL,
    FirstName nvarchar(25) NOT NULL,
    LastName nvarchar(25) NOT NULL,
    Age tinyint NULL
)

// Querry üzerinden Tablo Düzenleme
ALTER TABLE Employees
ADD [Description] nvarchar(250) NULL // Kolon ekleme
