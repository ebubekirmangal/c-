Form- Form Etiketleri (Input)

form: Kullanıcıdan veri almak için kullanılır. Bu etiketin içindeki form elementleri data submit yani veri gönderilceği zaman form etiketinin içindeki veriler gönderilir. Tek başına bir işe yaramaz.
input: Kullanıcının veri girişi yapabileceği giriş alanı tanımlar. attribute(parametre)'ü vardır. Type parametresi ile input türünü seçebilirsiniz. Id veta name parametresi ile label ve input arasındaki bağı oluşturursunuz, labeldaki for parametresindeki değerle inputdaki değer aynı ise bağ oluştururlar.Name parametresi her input elementinde bulunması gereken bir parametredir eğer alınan verileri aktarmak için kullanılır. Gönderilen verilerin hangi name altında toplanacağını belirtir.
label: İnput başlığı vermek için kullanılır. For parametresi input ile aradaki bağı kurmaya yarar.
select: Açılır liste oluşturmak için kullanılır.
option: Açılır liste içindeki elemanları oluşturmak için kullanılır.
textarea: Büyük metinler için kullanılır. Örneğin Hakkınızda metni oluşturmak için kullanılabilir. Metin satır ve sütun büyüklüğünü cols,rows parametresi ile oluşturabilirsiniz.
datalist: datalist aynı select gibi oluşturulur tek farkı değeri dataliste girilir inputta gösterilir . Örneği en sonki inputtur.(adı Liste, onun üstündeki select örneğidir.)

Forms Input Attributes- Formlarda Input Parametreleri

placeholder: input içerisinde gri tonda örnek yazı metni girmek için kullanılır.Örnek ilk örnekde placeholder yok ikinice vardır..

    

readonly: Input içine atılan value değerinin sadece okunmasını sağlar, değer değişikliği yapmamızı engeller. Örnek;




disabled: Readonly attribute ü ile benzer ama readonlydeki gibi etkileşim olmuyor direkt kapalı oluyor.




size: Input içerik uzunluğunu ayarlamak için kullanılır. Yukarda yaptığım input örneklerinin içerik uzunluk kısmı farklı.
min,max: Input type'ı pincode, number vb. (sayı girilebilen) tiplerde minimum ve maksimum değer aralığını vermek için kullanılır. Metinlerde ise minlength,maxlength olarak geçer ve minimum,maksimum karakter sayısı atamasını yapabilirsiniz.
required: Bu değer atanan inputlarda tipine göre değer girmek veya değer girmek zorunludur .Form etiketlerinde yaptığım örnekte göndere bastıktan sonra zorunlu olarak maili doldurmanızı istiyor.
autofocus: Sayfa açıldığında direkt inputun açılmasını sağlar.
autocomplete: Input'a tıkladığınızda daha önceden girdiğiniz verilerin çıkıp çıkmamasını bu parametre ile ayarlıyorsunuz.(on-off diyerek)
